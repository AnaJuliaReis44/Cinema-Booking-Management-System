# Cinema Booking Management System

## Overview

This project was developed for the QHO543 module. The system uses Python, Tkinter and SQLite to manage cinema bookings, customers, movies and payments.

## Features

* Relational database in SQLite
* Primary and Foreign Keys
* Triggers
* Views
* XML data storage
* BLOB image storage
* Python Tkinter GUI
* Customer, movie and booking management

## Technologies Used

* Python
* Tkinter
* SQLite
* SQLiteStudio
